03. 제이쿼리 이벤트(event)

웹문서에서 버튼을 클릭한다거나 마우스를 움직이거나 하는 모든 행위를 이벤트라고 한다. 그리고 이벤트 발생시 함수의 실행문이 수행 되도록 이벤트를 지정하는 것을 '이벤트 핸들러' 라고 한다.

1. 이벤트의 종류

  • 마우스 이벤트
  • 키보드 이벤트
  • 포커스 이벤트
  • 폼양식 이벤트
  • 윈도우 이벤트

2. 이벤트 등록방법(작성법)

  • 단독이벤트 - $('이벤트대상').이벤트메서드(function(){실행할내용});
  • 그룹이벤트 - $('이벤트대상').on('이벤트종류1','이벤트종류2'...,function(){실행할내용});

3. 마우스 이벤트와 폼양식 이벤트

  1. click - 마우스로 대상을 클릭했을 때 이벤트 내용이 실행
  2. dbclick - 마우스로 대상을 더블클릭했을 때 이벤트 내용이 실행
  3. mouseenter - 마우스로 대상을 오버했을 때 이벤트 내용이 실행(=mouseover)
  4. mouseleave - 마우스로 대상을 오버했다가 뺐을 때 이벤트 내용이 실행(=mouseout)
  5. hover - 마우스로 대상을 오버했을 때와 뺐을 때 둘다 이벤트 내용이 실행
  6. focus - 마우스로 클릭해 포커스가 발생되었을때 이벤트 발생(양식에서)
  7. blur - 폼양식에서 포커스가 해제되었을 때 이벤트가 발생. focusout과 유사
  8. toggle - 대상요소를 클릭할 때 마다 지정된 함수를 호출해 실행함
  9. each - 개별(각각에 대해)처리하는 이벤트 함수
