02. 자바스크립트 변수의 개념과 활용

변수(variable)란? 변하는 데이터를 저장하기 위한 일종의 메모리 공간이다.

변수에 저장된 값은 항상 마지막에 저장된 값이 출력이 된다.

1. 변수의 종류

  • 전역변수(global) : 문서전체에 영향을 주는 변수
  • 지역변수(local) : 함수(fuction)안에서 선언되어 사용되는 변수. 함수내에서만 사용 가능 (aka. 지역화폐)

2. 변수의 선언과 사용

변수를 작성할 때 'var', 'let', 'const' 키워드를 변수명 앞에 붙여 선언.

자바스크립트 최신버전인 es6에서는 'let', 'const'사용을 추천한다.

작성법 ) let 변수명; 또는 let 변수명=값;

3. 변수명 작성시 주의사항

  • 변수명은 반드시 영문 대, 소문자로 작성되어야 한다. 예) Aa01, aA01
  • 변수의 첫글자는 숫자, 기호가 올 수 없다. 예 ) 01A, 01_Aa
  • 변수명 사이에는 공백을 사용할 수 없다. 예 ) count 01
  • 변수명은 색상이 변하는 단어(예약어)는 사용할 수 없다. 예 ) var if, break등 교재참고
  • 변수명에는 특수기호를 사용할 수 없다.(단, '$'와 '_'는 사용가능) 예 ) !Aa, _Aa
  • 변수명은 의미를 담을 수 있는 이름으로 지정하고, 일명 낙타봉표기법(카멜표기법)으로 작성할 것을 권장한다. 예 ) imagename(X), imageName(O)
